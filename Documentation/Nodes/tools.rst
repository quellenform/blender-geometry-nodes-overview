Node-Based Tools
================

With version 4.0, geometry nodes can now also be created as tools, which are then available in the main menu.

+---------------+-----------------------------------+
|               |               4.0+                |
+===============+===================================+
| 3D Cursor     | `Yes <tools-3d-cursor-4.0_>`_     |
+---------------+-----------------------------------+
| Face Set      | `Yes <tools-face-set-4.0_>`_      |
+---------------+-----------------------------------+
| Selection     | `Yes <tools-selection-4.0_>`_     |
+---------------+-----------------------------------+
| Set Face Set  | `Yes <tools-set-face-set-4.0_>`_  |
+---------------+-----------------------------------+
| Set Selection | `Yes <tools-set-selection-4.0_>`_ |
+---------------+-----------------------------------+
